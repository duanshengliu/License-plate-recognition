# License-plate-recognition
目前只能识别蓝色车牌，还有一点缺少省份训练集，省份训练集只有['京', '闽', '粤', '苏', '沪', '浙']6个，字母和数字是都全的。
尽量保证识别的车牌图片不是很歪的一般可以识别，有的图片受拍摄光线、角度、或者清晰度的影响可能识别不了，因为还不是很完全的版本，有时间改进吧。
小伙伴有什么问题或者意见可以提提，能star就更好啦，谢谢啦！
